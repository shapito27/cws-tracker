## Tips for clean screenshots

- Capture at a consistent width (≈1200px) so images line up visually in the README.
- Prefer light mode for legibility on both GitHub light and dark themes.
- PNG is preferred for crisp UI text; keep each file well under ~1&nbsp;MB if possible
  (compress with e.g. [squoosh.app](https://squoosh.app) or `pngquant`).
- Avoid capturing any real API keys, proxy URLs, or private project data.
